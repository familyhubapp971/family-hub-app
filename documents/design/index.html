Family Hub — User Flow Board

High-fidelity screen mockups for every persona, arranged as a flow. Built with the live design system tokens from @familyhub/ui so what you see here matches what ships.

How to read this

  • One file per persona. Click in to walk that user's flow screen-by-screen.
  • Every screen is a real UI mockup, not a wireframe. Realistic copy, real components.
  • Backlog gap stickers mark screens that aren't designed or ticketed yet.
  • Ticket badges (e.g. FHS-25) link the screen to the Jira work that builds it.

Why this board exists

Engineering tickets describe what we're building. This board shows what users will actually see — so you can spot whether we're solving the right problem before the code lands.

If a screen here doesn't match what's planned, the ticket is wrong. If a screen has a backlog gap sticker, we have unticketed work that will block a real user.
