# Bugs — backlog

Lightweight backlog of known bugs that aren't yet ticketed in Jira.
Each file is one bug. Promote to a Jira FHS ticket (and delete the
file) when picked up.

Format:

- `<short-slug>.md`
- Front-matter: status (`open` / `in-jira: FHS-XXX` / `fixed`), date,
  found-by.
- Body: what happens, what should happen, repro, suspected cause,
  blast radius. Plain language at the top, technical bullets below.
